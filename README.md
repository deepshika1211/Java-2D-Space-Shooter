# 🚀 Space Shooter Game

A 2D arcade-style Space Shooter game developed using **Java Swing** and **Object-Oriented Programming (OOP)** concepts. The player controls a spaceship, destroys incoming enemies, earns points, and survives as long as possible with limited lives.

## 🎮 Features

- Player-controlled spaceship
- Movement using Arrow Keys
- Shoot bullets using Spacebar
- Random enemy spawning
- Bullet and enemy collision detection
- Score tracking system
- 3-Life gameplay system
- Game Over screen
- Restart functionality
- Built using Java Swing GUI
- Object-Oriented Design

## 🛠 Technologies Used

- Java
- Java Swing
- AWT Graphics
- Object-Oriented Programming (OOP)

## 🎯 Controls

| Key | Action |
|------|--------|
| ⬅ Left Arrow | Move Left |
| ➡ Right Arrow | Move Right |
| Spacebar | Shoot |

## 📷 Gameplay

The player controls a spaceship positioned at the bottom of the screen. Enemies spawn from the top and move downward. Shoot enemies before they reach the bottom of the screen. Each destroyed enemy increases the score. The player starts with 3 lives. The game ends when all lives are lost.
